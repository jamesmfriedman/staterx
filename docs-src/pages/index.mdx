# StateRx

## StateRx is a lightweight state management library built for RxJS.

StateRx leverages the power of RxJS to provide the basic primitives required for state management. It is un-opinionated about how you setup and organize your state and will work with any framework or vanilla js project you want to use. The only hard requirement is RxJS.

- Easy setup
- Use local or global state
- Optionally use Flux architecture
- Works with all frameworks
- Ships with hooks for React
- Can connect with redux devtools plugin (alpha)
- Out of the box Typescript support
- Requires RxJS 5.5 and up

## Get Started

`npm i staterx rxjs`
or
`yarn add staterx rxjs`
